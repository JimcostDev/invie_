
SHA-1: Es una función criptográfica hash que toma una entrada y produce un valor hash de 160 bits 
(20 bytes) conocido como resumen de mensaje, generalmente representado como un número hexadecimal, 
40 dígitos de largo.

--ver todos los commits
git log

--resumir commits
git log --oneline

--grafico, historia del proyecto
git log --oneline --graph

--elegir cantidad de commits a mostrar
git log -5
git log -4
git log -3
git log -2
git log -1
